== Getting Started

1. clone:
       git clone git@github.com:david-angga/test_chat.git

2. install juggernaut:
       npm install juggernaut

3. install redis-server:
       apt-get install redis-server

4. bundle
       bundle install
       
5. start juggernaut
       juggernaut

6. start redis
       sudo /etc/init.d/redis-server start
       
7. start rails
       rails s
